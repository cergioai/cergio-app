(1) SHARE MESSAGE → ONE-LINE
Tarik: "abbreviate or just keep initial few lines. no address (the
recipient will see all but user forwarding doesn't)". Trimmed to:

  "Hey — anyone know a good house cleaner?
   → http://localhost:5173/?ref=…"

Was previously a 3-line block w/ when + address + budget. The
forwarded text was too intrusive in iMessage / WhatsApp threads. The
recipient still sees full context on the linked page — no info lost.

buildShareMessage keeps the {when,where,budget,details} args in its
signature so all call sites work unchanged; the args are now no-ops
(void'd). Easy to re-include later if we want a flag.

(2) ORGANIC LOGO — "looks plastic, make it organic even if imperfect"
8 identical petals scaling in unison felt geometric. Rewrote BudBloom
with hand-tuned asymmetry:
  • Each petal: own rx (9–13), own ry (28–34), own opacity (0.78–0.95),
    own slight rotation nudge off the 45° grid (1–4° wobble)
  • Each petal animates INDEPENDENTLY with its own delay (0.00s–0.30s)
    so the bloom opens petal-by-petal instead of as one sprite
  • New keyframe cgBudPetal: scale 0.08 → 0.98 → 1.03 with a small
    rotate(-2deg → 1.5deg) wobble each cycle so the silhouette never
    lands in the exact same spot
  • Heart pulses on its OWN slower 5s rhythm (cgBudHeart, 0.96–1.12),
    offset from the 4s petal cycle — two breathing layers, not one
  • Added pale-orange smear ellipse inside the heart for hand-painted
    feel; replaces the flat #FFFFFF dot with FBCBA6/FFF combo

(3) IG SPOTLIGHT LINK — DISTINCT VISUAL LANE

its OWN region so the eye can find it without it shouting.
