1. URGENCY-WORD CAPTURE — "plumber asap" no longer asks "when?"

Local capture in applyParseResult: when fields.when is empty and
userInput contains an urgency word, set fields.when="ASAP" + mark
urgency=true. Pattern matches the bare-number budget fix.

  URGENCY_RE = /\b(asap|a\.s\.a\.p\.|now|right\s+now|right\s+away|
                 immediately|urgent(?:ly)?|emergency|today|tonight|
                 this\s+morning|this\s+afternoon|this\s+evening)\b/i

  "plumber asap"          → when="ASAP", urgency=true
  "plumber today"         → when="today", urgency=true
  "need plumber tonight"  → when="tonight", urgency=true

3. LOGO VARIANTS — switch between 4 organic marks behind one constant

Tarik picking between sprout / growth-rings / bud-bloom / pollen-pulse.
LeafLogo now dispatches via a LOGO_VARIANT constant. Each variant
animates ONLY when working=true (search-triggered, splash hero, etc.).

  const LOGO_VARIANT = 'rings';  // 'sprout' | 'rings' | 'bud' | 'pollen'
