PDP "Submit a request" sheet (RequestQuoteSheet.jsx)
- Tarik: "submit a request should open a request box... to the
  specific service (when clicked on on service profile)... can you
  same homepage box (and same logic (what where etc..)... it can also
  offer ... the ability to cross post that to all services at the end
  post submission".
- New component RequestQuoteSheet — modal sheet that opens from the
  "Don't see what you need?" callout on the PDP. Pre-fills:
    what  ← service.taxonomy_provider_type
    where ← default address (if signed in)
- Two-step flow:
    step 1 — compose: what / where / when / budget / notes. Submit
    calls api.createRequestToProvider() (new) — writes a requests row
    + a single notification to that provider's owner_id.
    step 2 — sent: "Sent to {Name}. Also notify other matching
    providers nearby?" → calls api.crossPostRequest() (new) which
    fans the SAME request_id to all other matching providers in
    range (same thread, more options).
- Replaces the prior navigate('/home') jump from the callout.
