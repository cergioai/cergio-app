══════════════════ (1) Activity red-dot — unread cue ═══════════════════

  New hook: src/hooks/useActivityUnread.js

    • localStorage `cergio:lastActivitySeenAt` stamped on every
      ActivityScreen mount.
    • Hook fetches listSocialFeed once + polls every 90s. Compares
      each event's `created_at` to the lastSeen timestamp.
    • Returns true if any fresh event landed since the user's last
      visit. Auto-suppresses while the user is on /activity.

  BottomNav:
    • Reads useActivityUnread when signed in.
    • Renders a small red dot (with white ring) on the Activity icon
      when fresh events exist AND the user isn't already on /activity.
    • Hidden for signed-out users (gated-tab lock takes precedence).

  Cheap, cookie-free, no read_state table — server-friendly.
