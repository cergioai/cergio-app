Logo motion v6 — Claude-style super-fun fluid morph (LeafLogo.jsx + index.css)

  Tarik (across two messages): "try to copy the motion of Claude's logo
  when animated.. make it infinitely more fluid... playful random" →
  "animate logo on search with super fun clever playful random
  patterns.. borrow from Claude's logo".

  Three independent layers now compose every cycle, all on coprime
  tempos so the silhouette never matches itself:

  Layer 1 — per-leaf BREATH + DRIFT (kept from v5)
    Continuous scale (0.94 ↔ 1.06) + sub-pixel translate (±1px) +
    rotation sway (±2.5°). Composed via the CSS Transforms Module
    Level 2 individual props (`scale`, `translate`, `rotate`) so
    they STACK instead of overriding each other. 4 breath templates
    × 2 drift templates × 7 jittered phase offsets.

  Layer 2 — WHOLE-MARK SLOW SPIN (new in v6)
    Outer wrapper rotates the entire flower around (60,60) on a 24s
    revolution. Heart counter-rotates on 38s. The two coprime spins
    make the geometry continuously evolve even when the per-leaf
    transforms are mid-pose.

  Layer 3 — PER-LEAF SHIMMER (new in v6)
    Each leaf's opacity flickers 0.72 ↔ 1.0 ↔ 0.86 on its own
    coprime 5-9s rhythm. Light catches one petal at a time and
    ripples around the flower — Claude's "thinking sparkle" tell.
    Composed via the `opacity` property so it doesn't fight the
    breath/drift transforms.

  Heart pulse upgraded to a 5-step fluid path on 7.3s using the same
  individual-property approach so the center breathes with the same
  character as the leaves.

  All animation only fires when `working` is true. At rest the flower
  is the still full-round bloom from v5.
