═══════════════ (1) HomeScreen i-icon ═══════════════

The (i) was wrapping below the Free-for-Connectors toggle because
the parent <div ref={modeBtnRef}> had no flex; two block-level
buttons stacked vertically. Wrapping it as `flex items-center`
pulls them onto a single row.

  • Toggle + (i) now sit inline + vertically aligned on the same
    row.
  • Hover tooltip — pure CSS via `group` + `group-hover`. Black/85
    bubble above the icon: "Creators, influencers, super-users
    with strong local networks. Tap for details."
  • `title` attribute carries the same string as a touch fallback
    (hover doesn't exist on phones).
  • Click still routes to /rainmaker/apply for the full pitch.
  • One icon serves both intents — find (consumer→service) AND
    spotlight (service→connector) — because freeServices toggles
    on both flows. No parallel surface needed.
